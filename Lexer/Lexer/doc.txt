Токены:
ERROR - токен ошибки

KEYWORD - for, const, class, public, private, string, int, char, bool, float, return
BOOLEAN - true, false

NUMBER - число - 1, 0, 12 и т.д, числа типа 01231234, 0000312313... - ошибка
IDENTIFIER - любая последовательность букв и цифр с буквы - a1 - идентификатор, 1a - error, _ не допустимы в id, max длина = 255
CONDITION_IF - токен для if
CONDITION_ELSE - токен для else
COMPARATOR - все виды сравнений - ==, !=, <=, >=, <, >
ASSIGNMENT - присваивание - =
FIXED_FLOAT - число вида 12.5, ??под вопросом 12. и .12??
FLOAT - число вида 321.12E-10, экспонента = E, допустимы .5E+12, 2.E-12, 3.3E+12, 7E+10
STRING - последовательность чего угодно между "" - "sdasda\n - ошибка, dasdsada" - ошибка, "sdasdads - ошибка и id, , max длина = 255
DELIMITER - разделители - ;.,
OPERATOR - операторы - +, -, *, /, составных операторов вроде += нет

OPEN|CLOSE_BRACKET - ()
ARRAY_OPEN|CLOSE - []
BLOCK_+OPEN|CLOSE - {}

CHAR - один символ между '', 'a' - ok, 'asd' - err, '' - err
BOOLEAN - токен для true/false
LOCGICAL_STATEMENT - and|or|not - ну думаю тут всё ясно, это вместо && || !

Никакого экранирования типа \n нет,
